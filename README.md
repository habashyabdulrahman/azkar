# تطبيق الأذكار (Azkar Web App)

تطبيق ويب تفاعلي وخفيف لعرض أذكار الصباح والمساء، مصمم بواجهة مستخدم بسيطة مع نظام مدمج لتتبع عدد مرات التكرار لكل ذكر.

## 🚀 المميزات (Features)

* [cite_start]جلب محتوى الأذكار ديناميكياً من واجهات برمجة التطبيقات (APIs)[cite: 1].
* [cite_start]توليد كروت الأذكار وعرضها في واجهة المستخدم تلقائياً باستخدام الجافاسكريبت (`.map()`)[cite: 2].
* [cite_start]عداد تفاعلي لكل ذكر مجهز بزر للزيادة وزر لإعادة الضبط (Reset)[cite: 3, 5].
* [cite_start]إيقاف زر العداد تلقائياً (تغيير المؤشر والشفافية) بمجرد الوصول للحد الأقصى لتكرار الذكر[cite: 5, 6].
* [cite_start]إعادة تفعيل العداد وتصفيره بسلاسة عند الحاجة[cite: 7].

## 🛠️ آلية العمل (Technical Overview)

1. [cite_start]**جلب وعرض البيانات:** يتم قراءة مصفوفة الأذكار وتمريرها لإنشاء هيكل HTML لكل كارت يحتوي على نص الذكر، الوصف، وعدد التكرار المطلوب[cite: 1, 2].
2. [cite_start]**إدارة الأحداث (Event Listeners):** بعد اكتمال تحميل الكروت في الـ DOM، يتم ربط أزرار التفاعل (الزيادة والإعادة) لكل كارت على حدة[cite: 4, 5].
3. **التحكم في الحالة (State Management):** يتم مقارنة العداد الحالي (`count`) بالحد الأقصى للذكر (`maxCount`). [cite_start]عند تطابقهما، يتم تعطيل زر الزيادة برمجياً لإعلام المستخدم بانتهاء هذا الذكر[cite: 5, 6].
